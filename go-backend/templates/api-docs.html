API Endpoints

üìÅ Workspaces

GET /api/v1/workspaces
List all workspaces for authenticated user
POST /api/v1/workspaces
Create a new workspace
GET /api/v1/workspaces/:id
Get workspace details by ID
PATCH /api/v1/workspaces/:id
Update workspace
DELETE /api/v1/workspaces/:id
Delete workspace

üìä Data Tables

GET /api/v1/tables?workspace_id={id}
List all tables in workspace
POST /api/v1/tables
Create a new data table
GET /api/v1/tables/:id
Get table with columns and metadata
GET /api/v1/tables/:id/rows
Get all rows in table
POST /api/v1/tables/:id/rows
Create a new row
PATCH /api/v1/tables/:id/rows/:row_id
Update row data
DELETE /api/v1/tables/:id/rows/:row_id
Delete row

üìù Forms

GET /api/v1/forms?workspace_id={id}
List all forms in workspace
POST /api/v1/forms
Create a new form
GET /api/v1/forms/:id
Get form with fields
POST /api/v1/forms/:id/submit
Submit form data
GET /api/v1/forms/:id/submissions
Get all form submissions

üéØ Request Hubs

GET /api/v1/request-hubs?workspace_id={id}
List all request hubs in workspace
POST /api/v1/request-hubs
Create a new request hub
GET /api/v1/request-hubs/:hub_id
Get request hub with tabs
GET /api/v1/request-hubs/by-slug/:slug
Get request hub by slug (requires workspace_id query param)
GET /api/v1/request-hubs/:hub_id/tabs
List all tabs in request hub
POST /api/v1/request-hubs/:hub_id/tabs
Create a new tab
POST /api/v1/request-hubs/:hub_id/tabs/reorder
Reorder tabs by position
