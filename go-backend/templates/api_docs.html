Workspaces

GET /api/v1/workspaces
List all workspaces. Query params: organization_id, is_archived
POST /api/v1/workspaces
Create a new workspace
GET /api/v1/workspaces/:id
Get workspace details with members, tables, forms, and hubs
PATCH /api/v1/workspaces/:id
Update workspace details
DELETE /api/v1/workspaces/:id
Soft delete a workspace

Request Hubs

GET /api/v1/request-hubs
List request hubs. Query param: workspace_id (required)
POST /api/v1/request-hubs
Create a new request hub
GET /api/v1/request-hubs/:hub_id
Get request hub details with tabs
GET /api/v1/request-hubs/by-slug/:slug
Get request hub by slug. Query param: workspace_id
PATCH /api/v1/request-hubs/:hub_id
Update request hub
DELETE /api/v1/request-hubs/:hub_id
Delete request hub
GET /api/v1/request-hubs/:hub_id/tabs
List all tabs in a request hub
POST /api/v1/request-hubs/:hub_id/tabs
Create a new tab in request hub
PATCH /api/v1/request-hubs/:hub_id/tabs/:tab_id
Update a tab
DELETE /api/v1/request-hubs/:hub_id/tabs/:tab_id
Delete a tab
POST /api/v1/request-hubs/:hub_id/tabs/reorder
Reorder tabs (batch update)

DELETE /api/v1/tables/:id

PATCH /api/v1/tables/:id/rows/:row_id
